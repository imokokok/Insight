# UI Types

This directory is reserved for UI-related TypeScript type definitions.

## Current State

The `src/types/ui/` directory currently contains only this README file. UI-related types are defined inline within component files or in other type locations as needed.

## Related Type Locations

UI component types are defined in the following locations:

- `src/components/ui/DataTablePro/types.ts` - DataTablePro column definitions and configuration types
- `src/components/ui/selectors/types.ts` - Selector component types (DropdownSelect, SegmentedControl)
- `src/components/export/types.ts` - Export component types
- `src/components/navigation/types.ts` - Navigation component types
- `src/components/search/types.ts` - Search component types
- `src/types/oracle/` - Oracle-related types (Blockchain enum, OracleProvider enum, PriceData, etc.)
- `src/types/api/` - API-related types (client, error, response types)
- `src/types/risk.ts` - Risk metrics types
- `src/types/guards.ts` - Type guard utilities
