# Checklist

## 环形图中心信息展示

- [ ] 中心显示总 TVS（大字号，如 "$27.6B"）
- [ ] 中心显示预言机数量（小字，如 "10 Oracles"）
- [ ] 中心显示当前时间范围（小字，如 "30D"）
- [ ] 悬停时中心切换显示选中预言机信息
- [ ] 悬停时显示：名称、份额百分比（大字号）、TVS、24h变化
- [ ] 鼠标移开后恢复显示总览信息
- [ ] 中心文字清晰可读，使用深色

## 环形图外围标签

- [ ] 在环形图周围显示预言机标签
- [ ] 标签包含：颜色点（w-2 h-2 rounded）+ 名称 + 份额百分比
- [ ] 标签位置对应扇形位置（顺时针排列）
- [ ] 标签使用小字号（text-xs）
- [ ] 名称使用灰色（text-gray-600）
- [ ] 份额使用深色（text-gray-900 font-medium）
- [ ] 标签悬停时对应扇形高亮
- [ ] 标签悬停时中心显示该预言机信息

## 扇形交互增强

- [ ] 悬停时扇形向外偏移（explode）5-10px
- [ ] 悬停时扇形放大（scale 1.05）
- [ ] 显示 tooltip（名称、份额、TVS、24h变化）
- [ ] 其他扇形降低透明度（opacity-50）
- [ ] 点击扇形选中该预言机
- [ ] 选中时扇形保持偏移状态
- [ ] 再次点击取消选中

## 扇形上直接显示份额

- [ ] 份额大于 5% 的扇形上显示百分比
- [ ] 使用白色文字，居中显示

- [ ] 文字清晰可读，不被颜色影响
