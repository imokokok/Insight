# Checklist

## 热力图交互
- [ ] 热力图悬停时显示价差金额、百分比和套利提示
- [ ] 热力图点击后价格趋势图自动聚焦对应两条链
- [ ] 热力图单元格有清晰的悬停和选中状态反馈

## 价格差异表格
- [ ] 表格上方有快速筛选按钮（全部/仅异常/仅正常）
- [ ] 异常值（价差超过阈值）使用醒目颜色高亮
- [ ] 每行显示价格变化趋势指示器（上升/下降箭头）
- [ ] 排序时基准链固定在首行

## 图表图例
- [ ] 双击图例可聚焦单链
- [ ] 聚焦状态有清晰的视觉反馈
- [ ] 再次双击可恢复全部显示

## 基准链选择
- [ ] 系统自动推荐流动性最高或请求量最大的链作为基准
- [ ] 显示推荐原因提示
- [ ] 基准链选择器中标注推荐链

## 数据刷新状态
- [ ] 刷新中显示加载指示器但不遮挡现有数据
- [ ] 刷新成功有简短提示
- [ ] 刷新失败显示错误提示和重试按钮
