# Checklist

## 布局结构优化
- [x] 左右两侧容器使用等高布局
- [x] 左侧价格趋势图表区域高度固定或自适应填充
- [x] 右侧三个卡片总高度与左侧相等
- [x] 左右两侧底部对齐无空缺

## 右侧卡片高度分配
- [x] 快速统计卡片高度适当，4个统计项分布均匀
- [x] 网络状态卡片高度适当，4个状态项分布均匀
- [x] 数据来源卡片填充剩余高度，4个数据源显示完整
- [x] 三个卡片之间间距一致

## 响应式布局

- [x] 移动端（lg以下）垂直堆叠布局正常
- [x] 各卡片内容无溢出或截断
- [x] 卡片内文字大小和间距自适应

## 视觉一致性
- [x] 卡片边框样式一致
- [x] 卡片圆角一致
- [x] 卡片内边距一致
