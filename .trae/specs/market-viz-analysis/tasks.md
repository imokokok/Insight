## 任务依赖关系

```
Task 3 (异常标注) 依赖 Task 1 (时间轴缩放) - 需要在缩放后的视图中标注异常
Task 5 (数据对比) 依赖 Task 1 (时间轴缩放) - 对比功能需要时间段选择
Task 7 (图表联动) 依赖 Task 6 (移动端优化) - 需要统一的筛选状态管理
```
