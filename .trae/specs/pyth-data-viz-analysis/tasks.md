# Pyth Network 数据可视化改进任务列表

本任务列表基于专业分析报告的改进建议制定，分为短期和中长期任务。

## 短期任务

### 任务 1: 接入真实 Pyth 数据源 ✅
**优先级**: 高  
**目标**: 替换所有 mock 数据，接入真实的 Pyth Network 数据

- [x] 调研 Pyth Network 官方 SDK (@pythnetwork/client)
- [x] 调研 Pyth Hermes API 用于获取实时价格流
- [x] 实现 PriceStream 组件的真实数据接入
- [x] 实现置信区间数据从 Pyth 价格账户解析
- [x] 实现发布者数据的实时获取
- [x] 添加数据缓存和错误处理机制

**已完成工作**:
1. 安装了 `@pythnetwork/hermes-client` 和 `axios` 依赖
2. 创建了 `src/lib/oracles/pythHermesClient.ts` 客户端，提供：
   - `getLatestPrice()` - 获取最新价格
   - `getHistoricalPrices()` - 获取历史价格
   - `subscribeToPriceUpdates()` - WebSocket 实时订阅
   - `getPublishers()` - 获取发布者数据
3. 更新了 `PriceStream` 组件，使用真实 Pyth 数据替代 mock 数据
4. 添加了加载状态和错误处理
