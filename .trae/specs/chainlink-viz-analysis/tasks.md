# Chainlink 数据可视化优化任务清单

本任务列表基于专业分析报告中的改进建议制定，按优先级排序。

---

## 高优先级任务

### Task 1: 简化价格图表默认显示状态 ✅
**目标**：减少默认视图中的视觉复杂度，提升可读性

- [x] 修改 PriceChart 默认设置
  - [x] 默认只显示价格线和 MA7
  - [x] 其他 MA 线（MA14/30/60/20）默认隐藏
  - [x] 标准差区间默认隐藏
  - [x] 预测区间默认隐藏
- [x] 更新图表设置持久化逻辑
- [x] 添加用户偏好提示（首次使用时提示可开启更多指标）

**验收标准**：
- 默认视图只显示价格线和一条MA线
- 用户切换指标后设置被正确保存
- 页面加载速度无明显下降

---

### Task 2: 实现图表时间联动功能 ✅
**目标**：让热力图与价格图表实现时间维度联动

- [x] 在 ActivityHeatmap 组件中添加点击事件
- [x] 实现时间范围上下文共享
- [x] 点击热力图某小时，价格图表跳转到对应时间段
- [x] 添加视觉反馈（高亮选中的时间块）

**验收标准**：
- 点击热力图任意时段，价格图表自动调整时间范围
- 联动过程有平滑过渡动画
- 移动端同样支持该功能

---

### Task 3: 优化移动端 KPI 展示 ✅
**目标**：减少移动端信息密度，提升可读性

- [x] 修改 KPIDashboard 移动端布局
  - [x] 每屏只显示 2 个 KPI 卡片
  - [x] 增加卡片间距
  - [x] 优化字体大小层级
- [x] 添加滑动手势支持
- [x] 优化滚动指示器视觉效果

**验收标准**：
- 移动端横向滑动流畅
- 卡片内容在 375px 宽度下完整显示
- 指示器清晰可点击

---

## 中优先级任务

### Task 4: 增强色盲友好性支持 ✅
**目标**：让红绿色盲用户也能准确理解数据

- [x] 价格涨跌增加形状区分
  - [x] 上涨使用向上箭头
  - [x] 下跌使用向下箭头
- [x] 图表线条增加样式区分
  - [x] MA 线使用不同虚线模式
  - [x] 价格线保持实线
- [x] 添加色盲模式切换选项

**验收标准**：
- 灰度模式下仍能区分涨跌
- 线条样式差异明显
- 色盲模式设置可持久化

---

### Task 5: 实现预言机数据对比功能 ✅
**目标**：支持 Chainlink 与其他预言机的数据对比

- [x] 创建 CrossOracleComparison 组件增强版
- [x] 支持同时选择 2-3 个预言机
- [x] 在同一图表中对比价格走势
- [x] 添加差异分析面板

**验收标准**：
- 支持 Chainlink、Pyth、Band 至少 3 个预言机对比
- 对比图表性能良好（数据点 >1000 时无卡顿）
- 差异数据实时计算

# 任务依赖关系

```
Task 1 (简化图表)
    │
    ├──> Task 3 (移动端优化) - 依赖简化后的图表设置
    │
    └──> Task 4 (色盲友好) - 可在简化基础上增强

Task 2 (图表联动)
    │
    └──> Task 6 (图表注释) - 共享事件系统

Task 5 (数据对比)
    │
    └──> Task 7 (导出优化) - 对比结果需要导出支持

Task 8 (技术债务)
    └── 无依赖，可随时进行
```
