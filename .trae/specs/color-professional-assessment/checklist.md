## 颜色可访问性评估

### 对比度检查
- [ ] 正文文本（14-16px）对比度 ≥ 4.5:1
- [ ] 大文本（18px+ 或 14px+粗体）对比度 ≥ 3:1
- [ ] 图表文本对比度 ≥ 4.5:1
- [ ] 按钮文本对比度 ≥ 4.5:1
- [ ] 链接文本对比度 ≥ 4.5:1

### 色盲友好性评估
- [ ] 不仅依赖颜色传递关键信息

- [ ] 考虑提供色盲友好的备选配色

- **A (优秀)**: 完全符合专业标准，无需改进
- **B+ (良好)**: 基本符合标准，有少量改进空间
- **B (一般)**: 部分符合标准，需要改进
- **C+ (需改进)**: 存在明显问题，需要重点改进
- **C (不合格)**: 不符合标准，必须立即改进
