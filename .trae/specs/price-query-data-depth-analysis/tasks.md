# 价格查询页面数据深度优化任务列表

## 任务概览

**目标**：在统计卡片和表格中添加 24 小时涨跌幅展示

- [x] SubTask 1.1: 扩展 PriceData 类型，添加 change24h 和 change24hPercent 字段
- [x] SubTask 1.2: 在 StatsGrid 组件中添加涨跌幅统计卡片
- [x] SubTask 1.3: 在 PriceResultsTable 中添加涨跌幅列
- [x] SubTask 1.4: 实现颜色编码（上涨绿色/下跌红色）

**验收标准**：
- 统计区域显示 24h 涨跌幅，带趋势图标
