# 数据导出功能差距分析 Spec

## Why

评估当前项目的数据导出功能与市场上成熟数据分析平台的差距，确定是否需要增强功能以保持竞争力。

## What Changes

本 spec 为分析文档，不涉及代码变更，主要对比分析以下方面：

### 当前实现的功能
- ✅ **CSV 导出**：支持当前价格和跨链数据的 CSV 格式导出
- ✅ **JSON 导出**：支持结构化 JSON 数据导出，包含元数据、当前价格、历史数据和摘要统计
- ✅ **时间戳文件名**：自动生成带时间戳的文件名，避免文件覆盖
- ✅ **前端导出**：使用 Blob 和 createObjectURL 在浏览器端完成导出
- ✅ **多页面覆盖**：在 cross-chain、price-query、cross-oracle 等主要页面实现导出功能

### 市场成熟平台的标准功能
根据市场调研，主流数据分析平台（如 Tableau、Power BI、Google Analytics、华为分析服务等）提供：

1. **多种导出格式**
   - ✅ CSV - 已实现
   - ✅ JSON - 已实现
   - ❌ Excel (XLSX) - 支持公式、多工作表、样式
   - ❌ PDF - 报表导出
   - ❌ PNG/SVG - 图表导出

2. **高级导出功能**
   - ❌ 异步导出任务 - 大数据量时后台处理
   - ❌ 导出进度显示 - 实时进度条
   - ❌ 导出历史 - 查看和重新下载历史导出文件
   - ❌ 自定义字段选择 - 用户选择要导出的列
   - ❌ 数据过滤后导出 - 仅导出筛选后的数据
   - ❌ 导出模板保存 - 保存常用导出配置

3. **企业级功能**
   - ❌ 定时自动导出 - 定期生成导出任务
   - ❌ API 导出接口 - 服务端导出 API
   - ❌ 大数据量优化 - 分块导出、流式导出
   - ❌ 导出权限控制 - 基于角色的导出权限
   - ❌ 审计日志 - 记录导出操作日志

## 竞争力评估

### 优势 ✓
1. **基础功能完备**：CSV 和 JSON 格式覆盖常用场景
2. **数据结构清晰**：JSON 导出包含元数据、摘要统计等完整信息
3. **用户体验良好**：按钮位置明显，禁用状态处理得当
4. **多语言支持**：导出内容使用国际化文本
5. **免费开源**：相比商业分析工具零成本

### 不足 ✗
1. **缺少 Excel 格式**：商业用户常用的 XLSX 格式不支持
2. **同步导出限制**：大数据量可能导致浏览器卡顿
3. **无导出历史**：用户无法查看之前的导出记录
4. **缺少自定义选项**：无法选择导出字段或应用筛选
5. **无服务端支持**：缺少后端 API 支持大规模数据导出

### 结论

**当前实现水平**：
- ✅ **个人开发者/小团队使用**：完全满足需求
- ⚠️ **中小企业使用**：基本满足，但大数据量场景有局限
- ❌ **企业级使用**：缺少企业级功能（异步导出、权限控制、审计日志等）

**对标结果**：
- 与 **基础版** 数据分析工具相当（如 Google Analytics 免费版、简单 BI 工具）
- 与 **企业版** 平台（如 Tableau Server、Power BI Premium）相比有明显差距
- 差距主要在**非功能性需求**：性能、可扩展性、管理功能

## ADDED Requirements

### Requirement: 增强导出功能（可选）
如果项目定位为企业级数据分析平台，系统 SHALL 提供：

#### 场景 1：Excel 导出
- **WHEN** 用户需要导出带格式的数据
- **THEN** 系统应支持 XLSX 格式，包含多工作表、列样式、冻结窗格等

#### 场景 2：大数据量导出
- **WHEN** 导出数据超过 10,000 条
- **THEN** 系统应自动切换为异步导出，显示进度提示，完成后通知用户

#### 场景 3：导出历史
- **WHEN** 用户查看导出历史
- **THEN** 系统应显示最近 30 天的导出记录，支持重新下载

#### 场景 4：自定义导出
- **WHEN** 用户在导出前
- **THEN** 系统应允许选择导出字段、应用筛选条件、保存导出模板

### Requirement: 保持现状（推荐）
如果项目定位为**轻量级、个人/小团队使用的分析工具**，当前功能已足够：
- 保持现有 CSV/JSON 导出功能
- 优化现有实现的性能和用户体验
- 暂不添加复杂的企业级功能

## 建议

基于当前实现和市场定位，建议：

1. **短期（当前版本）**

2. **中期（如需提升竞争力）**

   - 实现大数据量时的异步导出机制
   - 提供自定义字段选择功能

3. **长期（企业级定位）**
   - 构建服务端导出 API
   - 实现定时导出和推送功能
   - 添加权限控制和审计日志
